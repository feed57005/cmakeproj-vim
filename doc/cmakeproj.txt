COMMANDS                                               *cmakeproj-commands*

                                                       *cmakeproj-:BuildType*
:BuildType [type]         Sets build type.
                          Argument [type] can be 'debug' or 'release'.

                                                       *cmakeproj-:BuildTarget*
:BuildTarget [target]     Sets makefile target to build.

                                                       *cmakeproj-:RunCMake*
:RunCMake                 Runs CMake.


GLOBAL OPTIONS                                         *cmakeproj-options*

                                                       *cmakeproj_cmake_bin*
Default: "cmake"
Path to CMake command
                                                       *cmakeproj_generator*
Default: "ninja"
Generator to use when running CMake. Currently it can be 'ninja' or 'make'.
